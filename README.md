# Billing Analytics Team

Este repositório contém a estrutura e definições para uma equipe de Analytics focada em faturamento e faturamento complexo (Billing Analytics).

## Estrutura do Projeto

- `.agents/`: Definições de personas, personas, kb, skills e workflows para os agentes de faturamento.
- `docs/`: Documentação geral, incluindo templates de auditoria e regras de negócio.

## Personas e Agentes

A equipe é composta por:

- **@billing-pm**: Traduz regras fiscais e de faturamento em especificações técnicas.
- **@analytics-engineer**: Constrói modelos dbt/SQL modulares seguindo a arquitetura Medallion.
- **@financial-auditor**: Garante a integridade da cascata de faturamento e evita perda de receita.
- **@code-explorer**: Mapeia linhagem de dados e identifica impactos em ETLs legados.
- **@sql-optimizer**: Otimiza planos de execução e reduz custo computacional.

## Como utilizar

Esta estrutura foi desenhada para facilitar a colaboração e automação em processos de auditoria e engenharia de dados financeira.
